Summary: Latitude vs. X

Web Visualization Dashboard (Latitude)
This Project utilizes HTML, CSS and Bootstrap along with Python (Jupyter Notebook using pandas, matplotlib and numpy libraries) to analyze Cities data to find relationship between cities latitude Vs. Temperature, Humidity, Cloudiness and Wind Speed using Scatter Plots. The scatter plot reveals correlation (positive, negative or no correlation ) in a large amount of data.

Analysis :
Latitude Vs. Temperature - It is found that there is a positive correlation between Temperature and Latitude as one variable increases so does the other.
Latitude Vs. Humidity - It is found that there is a weak correlation between Humidity and Latitude as one variable increases the other decreases.
Latitude Vs. Cloudiness - It is found that there is a no correlation between Humidity and Latitude as one increases, the other one is not affected.
Latitude Vs. Wind Speed - It is found that Wind speed tends to be strong as one moves away from the equator.
