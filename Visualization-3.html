Web Visualization - Cloudiness Vs. Cities Latitude
We are using Scatter plot to understand Relationship between cities latitude and Cloudiness. The scatter plot is simply a set of data points plotted on an X and Y axis to represent two sets of variables. The shape those data points create tells the story, most often revealing correlation (positive, negative or no correlation ) in a large amount of data.

To plot the Scatter plot we are using Matplotlib, Numpy and Pandas (libraries of Python). The X axis is Latitude of Cities and Y axis is Cloudiness of that cities.
Analysis :
It is found that there is a no correlation between Humidity and Latitude. There is no apparent relationship between the variables as one increases, the other one is not affected.
